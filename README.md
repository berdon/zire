# Zire
Automate/streamline working amongst git and jira.

## Development
### Dependencies
- nodejs
- typescript

### Building
```bash
npm install
tsc
```

## Install
```bash
npm install -g
```

## Usage
```bash
zr ls               # List assigned issues
zr (st|status)      # Show current issue being worked on
zr (co|checkout)    # Start working on an issue
```
